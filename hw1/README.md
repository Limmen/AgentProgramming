# Homework 1 ID2209 - SmartMuseum Multi Agent System

## Commands

**build** 

`mvn install`

**run main container**

`./main.sh`

**run all agents same container**

`./agents.sh`

**run single curator agent on a container**

`./curator.sh`

**run single profiler agent on a container**

`./profiler.sh`

**run single tourguide agent on a container**

`./tourguide.sh`
